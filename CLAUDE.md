# Bioinformatics Research Project

## Stack
Python, pyDESeq2, scanpy, snakemake, jupyter

## Goals
- Replicate a published RNA-seq DEG analysis from GEO
- Cross-cohort validation in independent datasets
- Preprint-quality output on bioRxiv

## Rules
- Never skip FDR correction
- Always set random seeds
- All data downloaded via script (no manual downloads)
- Log checksums for every dataset
